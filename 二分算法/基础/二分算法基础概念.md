## 一、二分查找的本质

- **前提**：搜索空间具有**单调性**（或局部可二分性）。
- **核心思想**：每次将搜索区间缩小一半，时间复杂度 O(log⁡n)。
- **关键操作**：确定 `mid`、判断条件、移动左右指针。

竞赛中出错最多的是**死循环**和**边界遗漏**。掌握一套无死循环的固定模板最重要。

------

## 二、整数二分（两种经典模板）

### 2.1 模板选择原则

设 nums 为递增（非递减）数组，长为 n。

常用转化：

| 需求                     | 写法                        | 如果不存在 |
| :----------------------- | :-------------------------- | :--------- |
| ≥ x 的第一个元素的下标   | lowerBound(nums, x)         | 结果为 n   |
| > x 的第一个元素的下标   | lowerBound(nums, x + 1)     | 结果为 n   |
| < x 的最后一个元素的下标 | lowerBound(nums, x) − 1     | 结果为 −1  |
| ≤ x 的最后一个元素的下标 | lowerBound(nums, x + 1) − 1 | 结果为 −1  |

-

|        需求        | 写法                          |
| :----------------: | :---------------------------- |
| \(< x\) 的元素个数 | `lowerBound(nums, x)`         |
| (≤ x\) 的元素个数  | `lowerBound(nums, x + 1)`     |
|  (≥x\) 的元素个数  | `n - lowerBound(nums, x)`     |
| \(> x\) 的元素个数 | `n - lowerBound(nums, x + 1)` |

注意：\(< x\) 和 \(≥ x\) 互为补集，元素个数之和为 n。\(≤ x\) 和 \(> x\) 同理。

### 2.2 模板一：左闭右开 [ l , r )

python

```python
# lower_bound 返回最小的满足 nums[i] >= target 的下标 i
# 如果数组为空，或者所有数都 < target，则返回 len(nums)
# 要求 nums 是非递减的，即 nums[i] <= nums[i + 1]
    def lower_bound(self, nums: List[int], target: int) -> int:
        left, right = 0, len(nums)  # 左闭右开区间 [left, right)
        while left < right:  # 区间不为空
            # 循环不变量：
            # nums[left-1] < target
            # nums[right] >= target
            mid = (left + right) // 2
            if nums[mid] >= target:
                right = mid  # 范围缩小到 [left, mid)
            else:
                left = mid + 1  # 范围缩小到 [mid+1, right)
        # 循环结束后 left = right
        # 此时 nums[left-1] < target 而 nums[left] = nums[right] >= target
        # 所以 left 就是第一个 >= target 的元素下标
        return left
```



### 2.3 模板二：开区间 (left, right)

python

```python
 # lower_bound 返回最小的满足 nums[i] >= target 的下标 i
    # 如果数组为空，或者所有数都 < target，则返回 len(nums)
    # 要求 nums 是非递减的，即 nums[i] <= nums[i + 1]
    def lower_bound(self, nums: List[int], target: int) -> int:
        left, right = -1, len(nums)  # 开区间 (left, right)
        while left + 1 < right:  # 区间不为空
            mid = (left + right) // 2
            # 循环不变量：
            # nums[left] < target
            # nums[right] >= target
            if nums[mid] >= target:
                right = mid  # 范围缩小到 (left, mid)
            else:
                left = mid  # 范围缩小到 (mid, right)
        # 循环结束后 left+1 = right
        # 此时 nums[left] < target 而 nums[right] >= target
        # 所以 right 就是第一个 >= target 的元素下标
        return right     
```



### 2.4 统一闭区间模板（推荐通用写法）

使用闭区间 `[l, r]`，结构统一，不易写错。

python

```python
# lower_bound 返回最小的满足 nums[i] >= target 的下标 i
# 如果数组为空，或者所有数都 < target，则返回 len(nums)
# 要求 nums 是非递减的，即 nums[i] <= nums[i + 1]
def lower_bound(self, nums: List[int], target: int) -> int:
        left, right = 0, len(nums) - 1  # 闭区间 [left, right]
        while left <= right:  # 区间不为空
            # 循环不变量：
            # nums[left-1] < target
            # nums[right+1] >= target
            mid = (left + right) // 2
            if nums[mid] >= target:
                right = mid - 1  # 范围缩小到 [left, mid-1]
            else:
                left = mid + 1  # 范围缩小到 [mid+1, right]
        # 循环结束后 left = right+1
        # 此时 nums[left-1] < target 而 nums[left] = nums[right+1] >= target
        # 所以 left 就是第一个 >= target 的元素下标
        return left
```



> 求最后一个满足条件的位置时，只需将记录答案的逻辑放在 `else` 分支，并改为 `l = mid + 1` 时更新即可。

## 三、浮点数二分

在实数域上寻找满足精度的边界，通常采用固定迭代次数（避免精度麻烦）。

python

```python
def float_binary_search(l, r, check, iterations=100):
    """在 [l, r] 上二分，check 为单调判定函数，返回边界"""
    for _ in range(iterations):      # 迭代 100 次精度可达 1e-30 级别
        mid = (l + r) / 2.0
        if check(mid):
            r = mid
        else:
            l = mid
    return l  # 或 r，二者此时近似相等
```



- 若需精确控制，可将迭代改为 `while r - l > 1e-7:`，但固定迭代次数更稳妥。
- 常见应用：方程求根、几何计算、辅助三分法等。

## 四、二分答案

当问题的解具有**单调性**（若 `x` 可行，则所有 `> x` 或 `< x` 都可行），可以二分可能的答案，并用判定函数 `check(mid)` 判断可行性。

**模板（最小可行解）**：

python

```python
def min_feasible(low, high, check):
    """返回使得 check 为真的最小值，若全部为假返回 high+1"""
    l, r = low, high
    ans = r + 1
    while l <= r:
        mid = l + (r - l) // 2
        if check(mid):
            ans = mid
            r = mid - 1   # 尝试更小
        else:
            l = mid + 1
    return ans
```



- **最大化可行解**：将记录 `ans` 的分支放在 `else` 或调整判定逻辑即可，例如改成 `if check(mid): l = mid + 1, ans = mid`。
- 常见题型：最小化最大值、最大化最小值、分组划分、第 K 大/小等。

## 五、进阶技巧与变体

### 5.1 旋转排序数组的搜索

- **无重复值**：通过比较 `a[mid]` 与端点值确定有序区间，再判断目标在哪一侧。
- **有重复值**：最坏情况会退化为 O(n)*O*(*n*)，遇到 `a[l] == a[mid] == a[r]` 时收缩边界（`l += 1; r -= 1`）。

### 5.2 寻找峰值

- 利用“上坡必有峰”的性质：比较 `a[mid]` 和 `a[mid+1]`，若 `a[mid] < a[mid+1]` 则峰值在右侧，否则在左侧（含 `mid`）。

### 5.3 二分 + 交互

- 与评测机交互获取 `check` 结果。在 Python 中注意使用 `print(..., flush=True)` 或 `sys.stdout.flush()`。

### 5.4 避免死循环与溢出

- Python 整数无溢出，但养成 `mid = l + (r - l) // 2` 的习惯仍能保持一致性。
- **左闭右开**：`while l < r`，配合 `l = mid + 1` / `r = mid`。
- **左开右闭**：`while l < r`，配合 `l = mid` / `r = mid - 1`，且必须用右中位数 `(r - l + 1) // 2`。
- **闭区间**：`while l <= r`，配合 `l = mid + 1` / `r = mid - 1`，最不容易错。

### 5.5 Python 内置二分模块 `bisect`

python

```python
import bisect

idx_left = bisect.bisect_left(a, target)   # 第一个 >= target 的索引
idx_right = bisect.bisect_right(a, target) # 第一个 > target 的索引
# 若需判断存在性：idx_left != len(a) and a[idx_left] == target
```



- 手写模板适合需要自定义 `check` 函数的场景（如二分答案）。

## 六、常见错误检查清单

1. **死循环**：
   - 模板一（左中）不能写 `l = mid`，必须是 `l = mid + 1` 与 `r = mid`。
   - 模板二（右中）必须用 `mid = l + (r - l + 1) // 2`，配合 `l = mid`、`r = mid - 1`。
2. **区间定义不一致**：闭区间 `l <= r`；左闭右开 `l < r`；左开右闭需注意 `l = -1`。
3. **无解处理**：始终初始化 `ans` 为“无解”标志，如 `-1` 或列表长度。
4. **浮点数精度**：用固定迭代次数代替 `r - l > eps` 的判断。
5. **判定函数单调性错误**：二分答案前务必确认问题具有单调性。

------

## 七、总结

- **基础**：有序数组的 `bisect_left` / `bisect_right` 及手写模板。
- **核心**：单调布尔函数的边界查找，闭区间 `while l <= r` 并记录答案的写法最通用。
- **应用**：二分答案 + 判定函数，是解决最优化问题的利器。
- **进阶**：旋转数组、峰值、交互题、浮点二分、三分（求单峰极值）等。
