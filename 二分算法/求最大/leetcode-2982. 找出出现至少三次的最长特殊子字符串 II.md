# [2982. 找出出现至少三次的最长特殊子字符串 II](https://leetcode.cn/problems/find-longest-special-substring-that-occurs-thrice-ii/)

给你一个仅由小写英文字母组成的字符串 `s` 。

如果一个字符串仅由单一字符组成，那么它被称为 **特殊** 字符串。例如，字符串 `"abc"` 不是特殊字符串，而字符串 `"ddd"`、`"zz"` 和 `"f"` 是特殊字符串。

返回在 `s` 中出现 **至少三次** 的 **最长特殊子字符串** 的长度，如果不存在出现至少三次的特殊子字符串，则返回 `-1` 。

**子字符串** 是字符串中的一个连续 **非空** 字符序列。

**示例 1：**

```
输入：s = "aaaa"
输出：2
解释：出现三次的最长特殊子字符串是 "aa" ：子字符串 "aaaa"、"aaaa" 和 "aaaa"。
可以证明最大长度是 2 。
```

**示例 2：**

```
输入：s = "abcdef"
输出：-1
解释：不存在出现至少三次的特殊子字符串。因此返回 -1 。
```

**示例 3：**

```
输入：s = "abcaba"
输出：1
解释：出现三次的最长特殊子字符串是 "a" ：子字符串 "abcaba"、"abcaba" 和 "abcaba"。
可以证明最大长度是 1 。
```

**提示：**

- `3 <= s.length <= 5 * 105`
- `s` 仅由小写英文字母组成。

```python
class Solution:
    def maximumLength(self, s: str) -> int:
        def check(mid):
            cnt =defaultdict(int)
            i=0
            n = len(s)
            while i<n:
                j=i+1
                while j<n and s[j]==s[i]:
                    j+=1
                cnt[s[i]]+=max(0,j-i-mid+1) 
                i=j
            return max(cnt.values())
        left = 0
        right = len(s)
        while left+1<right:
            mid=(left+right)//2
            if check(mid)>=3:
                left = mid
            else:
                right = mid
        return -1 if left==0 else left
```



`max(0, j - i - mid + 1)` 这行代码的作用是：**计算一段连续相同字符中，能够提供多少个长度为 `mid` 的特殊子串**。

### 详细拆解

在 `check(mid)` 函数里，这段逻辑是：

python

```python
i = 0
while i < n:
    j = i + 1
    while j < n and s[j] == s[i]:
        j += 1
    # 此时从 i 到 j-1 是一段完全相同的字符，长度为 run_len = j - i
    cnt[s[i]] += max(0, j - i - mid + 1)
    i = j
```



- **`j - i`** 就是当前这段连续相同字符的长度（记为 `L`）。
  比如 `s = "aaabaaa"`，当 `i=0` 时，`j` 会走到 `3`，长度 `L = 3`。
- 对于长度 `L` 的完全由同一个字符 `c` 组成的字符串，它包含的**长度为 `mid` 的子串**（允许重叠）的数量是：
  - 如果 `L >= mid`：有 `L - mid + 1` 个。
    例如 `L=3, mid=2`，子串为 `s[0:2]` 和 `s[1:3]`，共 `3-2+1=2` 个。
  - 如果 `L < mid`：一个都没有，即 0 个。
- 用 `max(0, L - mid + 1)` 就统一了这两种情况，确保不会算出负数。

所以，`max(0, j - i - mid + 1)` 就是当前这一段相同字符能贡献的“长度为 `mid` 的特殊子串”个数。通过循环累加到 `cnt[s[i]]`，就能统计出这个字符在整个字符串中一共能提供多少个这样的子串。

### 示例

假设 `s = "aaabaaa"`，当前 `mid = 2`：

- 第一段 `"aaa"`（长度 3）：`max(0, 3 - 2 + 1) = 2`，给字符 `'a'` 加 2。
- 中间 `'b'` 单独一段（长度 1）：`max(0, 1 - 2 + 1) = 0`，不加。
- 第二段 `"aaa"`（长度 3）：再加 2。
- 最终 `cnt['a'] = 4`，表示长度为 2 的全 `'a'` 子串共出现 4 次。

### 算法思路

1. **二分答案**
   - 要找最大长度 `L`，使得存在某个字符，其在原字符串中长度为 `L` 的特殊子串出现至少 3 次。
   - 若长度为 `L` 满足条件，则所有小于 `L` 的长度也一定满足；反之，若 `L` 不满足，则更大的长度也不会满足。这种单调性可以用二分查找。
   - `left` 初始化为 0，`right` 初始化为 `len(s)`。循环 `while left + 1 < right`，最终 `left` 即为满足条件的最大长度，若 `left == 0` 则返回 -1。
2. **检验函数 `check(mid)`**
   - 判断是否存在一个字符，它能够提供至少 3 个长度为 `mid` 的特殊子串。
   - 将字符串按连续相同字符分段（run-length encoding）。对于长度为 `run_len` 的一段全为字符 `c` 的区间，它可以贡献 `max(0, run_len - mid + 1)` 个长度为 `mid` 的子串（全部由 `c` 组成）。
   - 用字典 `cnt` 累加每个字符的贡献次数，最后返回 `max(cnt.values())`，即所有字符中出现次数最多的那个字符的出现次数。
   - 若该最大值 ≥ 3，则 `mid` 可行。

### 复杂度分析

- **时间复杂度**：`check` 函数遍历一次字符串，耗时 O(n)；二分查找执行 O(log n) 次。总复杂度 **O(n log n)**，对于题目中 n ≤ 5×10⁵ 的规模是可行的。
- **空间复杂度**：只使用了常数个变量和一个小字典（键为字符，最多 26 个小写字母），**O(1)**。
