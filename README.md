## 算法分类概览

| 算法类型 | 核心思想 | 题目数量 | 模板文档 |
| :---: | :---: | :---: | :---: |
| 二分算法 | 每次将搜索区间缩小一半，O(log n) | 14 | 基础概念 / 求最大模板 / 求最小模板 |
| DFS / 回溯 | 沿一条路径深入探索，无法继续时回溯 | 2 | DFS基本概念 / 回溯基本概念 |
| 动态规划 | 将问题拆解为重叠子问题，记忆化求解 | 2 | 动态规划模板 |
| 单调栈 | 维护单调序列，O(n) 解决最近大于/小于问题 | 4 | 算法模板 |
| 枚举技巧 | 利用数据特征优化暴力枚举 | 4 | - |
| 贪心 | 每步选择局部最优，期望得到全局最优 | 1 | - |

## 笔记特点

- **模板先行**：每个算法类型都整理了对应的模板代码，方便快速套用
- **分类清晰**：按算法类型和难度分层组织（如二分算法分为基础、求最大、求最小）
- **详解思路**：每道题包含题目描述、思路分析和代码实现
- **覆盖双平台**：包含 LeetCode 和蓝桥杯题目
