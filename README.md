## 算法分类概览

| 算法类型 | 核心思想|模板文档 |
| :---: | :---: |  :---: |
| 二分算法 | 每次将搜索区间缩小一半，O(log n)|  基础概念 / 求最大模板 / 求最小模板 |
| DFS / 回溯 | 沿一条路径深入探索，无法继续时回溯|   DFS基本概念 / 回溯基本概念 |

| 单调栈 | 维护单调序列，O(n) 解决最近大于/小于问题|    算法模板 |
| 枚举技巧 | 利用数据特征优化暴力枚举|  - |
| 贪心 | 每步选择局部最优，期望得到全局最优|  - |
