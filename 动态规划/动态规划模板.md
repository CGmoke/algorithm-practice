## 一、如何在蓝桥杯中判断一道题是 DP？

蓝桥杯题目通常有明确的关键词和结构特征，遇到以下信号时优先考虑 DP：

### 1. 问法特征（最值 / 计数 / 可行性）

- **最值问题**：“最大 / 最小 / 最长 / 最短 / 最多 / 最少”
- **计数问题**：“一共有多少种方案 / 路径数 / 排列数”，且通常要求模 `1e9+7` 或取后几位
- **可行性问题**：“能否凑出 / 能否到达 / 是否存在一种选法”
- **概率期望**：出现较少，但也是 DP（蓝桥杯初期不太考）

### 2. 问题结构特征（子问题重叠、无后效性）

- 题目有明显的**阶段性**：一天天、一步步、一个个物品、一棵树的子树
- 决策依赖于**当前状态**，且当前状态一旦确定，后续决策不受之前具体路径影响（无后效性）
- **暴力枚举会指数爆炸**，但问题可以拆成“前面部分的最优解 + 当前决策”
- 数据范围：**n 通常在 1e2 ~ 1e5**（蓝桥杯省赛尤其喜欢 n=1e5 的线性 DP 或简单背包）

### 3. 蓝桥杯特有的“伪装”

- 题目表面是字符串、图、数学，但实际状态定义后就是 DP（例如“数字三角形”就是经典入门）
- **看起来是贪心但贪心不对**：这时几乎必是 DP（例如“不能只选性价比最高的物品”）
- 出题人喜欢用**迷宫、金币、日期、砝码、数字划分**等生活化场景包装

> 快速判断口诀：
> **求最优问方案数，阶段明显无后效，贪心不对暴力慢，十有八九是 DP。**

## 二、蓝桥杯 DP 的常用模板

下面按考察频率从高到低列出蓝桥杯核心 DP 模型，每种给出**状态定义、转移方程、代码骨架**。

### 1. 线性 DP（最高频）

**适用**：一维序列上的问题，如最大子段和、最长上升子序列、打家劫舍等。

**模板：最大子段和（连续）**

```python
dp = [0] * n
dp[0] = a[0]
for i in range(1, n):
    dp[i] = max(dp[i-1] + a[i], a[i])
ans = max(dp)
```



**模板：最长上升子序列 (LIS)**

```python
# O(n^2) 做法，蓝桥杯很多时候 n 不大，直接写
dp = [1] * n
for i in range(n):
    for j in range(i):
        if a[j] < a[i]:
            dp[i] = max(dp[i], dp[j] + 1)
ans = max(dp)
# 若 n 较大 (>=1e4)，再用二分优化
```



**模板：不相邻选取最大值（打家劫舍）**



```python
dp = [0] * n
dp[0] = a[0]
if n > 1:
    dp[1] = max(a[0], a[1])
for i in range(2, n):
    dp[i] = max(dp[i-1], dp[i-2] + a[i])
```



### 2. 背包 DP（必考）

蓝桥杯每年必出变种背包。要能区分 0-1、完全、多重、分组。

**0-1 背包**（每个物品最多选一次）



```python
dp = [0] * (V + 1)
for w, v in items:          # w 重量 / 体积，v 价值
    for j in range(V, w - 1, -1):
        dp[j] = max(dp[j], dp[j - w] + v)
```



**完全背包**（每个物品无限次）



```python
for w, v in items:
    for j in range(w, V + 1):   # 正序就是无限次
        dp[j] = max(dp[j], dp[j - w] + v)
```



**方案数背包**（如凑零钱、砝码称重）



```python
dp = [0] * (target + 1)
dp[0] = 1
for num in nums:                # 物品
    for j in range(target, num - 1, -1):   # 0-1 或完全根据题意
        dp[j] += dp[j - num]
```



### 3. 区间 DP

**特征**：问题关于一个区间，每次操作合并或拆分两端，如石子合并、回文串。

**模板：石子合并（最小值）**



```python
dp = [[0] * n for _ in range(n)]
# 需要前缀和 pre
for length in range(2, n + 1):          # 枚举区间长度
    for l in range(0, n - length + 1):
        r = l + length - 1
        dp[l][r] = float('inf')
        for k in range(l, r):
            dp[l][r] = min(dp[l][r], dp[l][k] + dp[k+1][r] + pre[r+1] - pre[l])
```



### 4. 树形 DP

**特征**：数据结构为树，通常问“在子树中选择/不选择”的最大价值。

**模板：树上最大独立集（没有上司的舞会）**



```python
def dfs(u, parent):
    dp0[u] = 0    # 不选 u
    dp1[u] = val[u]  # 选 u
    for v in adj[u]:
        if v == parent: continue
        dfs(v, u)
        dp0[u] += max(dp0[v], dp1[v])
        dp1[u] += dp0[v]
# 答案 = max(dp0[root], dp1[root])
```



### 5. 状态压缩 DP

**特征**：n 很小（<= 20），状态涉及集合（选/没选）。

**模板：TSP / 覆盖问题**



```python
dp = [[inf] * n for _ in range(1 << n)]
dp[1][0] = 0   # 从 0 出发
for mask in range(1 << n):
    for u in range(n):
        if mask & (1 << u):
            for v in range(n):
                if not mask & (1 << v):
                    dp[mask | (1 << v)][v] = min(dp[mask | (1 << v)][v], dp[mask][u] + dist[u][v])
```



### 6. 数位 DP

**特征**：求区间 [L, R] 内满足某种条件的数字个数（如不含 4，数字之和等），L, R 极大。

**模板框架（记忆化搜索）**

python

```python
def dfs(pos, tight, ...):
    if pos == len(digits): return 满足条件则 1 否则 0
    if not tight and dp[pos][...] != -1: return dp[pos][...]
    limit = digits[pos] if tight else 9
    res = 0
    for d in range(limit + 1):
        res += dfs(pos+1, tight and d==limit, ...)
    if not tight: dp[pos][...] = res
    return res
```

## 三、DP 解题四步法（蓝桥杯实战流程）

1. **定状态**
   问自己：什么变量能唯一描述当前子问题？常见：`dp[i]` 表示前 i 个元素的结果；`dp[i][j]` 表示前 i 个物品、容量 j 时的最优值。
2. **推转移**
   对于当前状态，有哪些决策？如何由已知状态得来？先写出文字递推关系再翻译成代码。
3. **初始化**
   无意义的起点、边界值（0 或 `-inf`）。特别注意计数类 DP 的 `dp[0] = 1`。
4. **优化（蓝桥杯够用就行）**
   首先保证正确，再考虑滚动数组、维度压缩（如背包倒序）。复杂度在 1e7 以内一般都能过。

## 四、蓝桥杯常见坑点

- **题目要求的模数**：计数类 DP 每步都要取模。
- **数组维度定义**：长度多开 1～2 格，避免越界。
- **0-1 背包的倒序**：完全背包正序，搞反会失去限制。
- **答案不一定是 `dp[n]`**：有时是 `max(dp)` 或 `dp` 中某个特殊状态。
- **初始化为负无穷**：求最大值用 `-inf`，最小值用 `inf`。
